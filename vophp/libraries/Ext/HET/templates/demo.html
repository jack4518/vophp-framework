注意事项

HET的解释语体中，在单引号中使用单引号，必须将单引号\转义
例如
\{$abc = '$_ABC['abc']'\} 错误
\{$abc = '$_ABC[\\'abc\\']'\} 正确
\{$abc = '$_ABC["abc"]'\} 正确
\{$dog = 'It's a dog'\} 错误
\{$dog = 'It\\'s a dog'\} 正确


务必注意，在语体中使用{}必须将其转义。
在语体将{}转义的方法是使用\\{和\\}
比如：
\{$abc= '\\{asdfa\\}'\}{$abc= '\{asdfa\}'}将会输出结果：{$abc}
变量输出

语法: {变量名}

已定义变量：\{$title\}　　　 输出：{$title}
未定义变量：\{$name\}　　　 输出：{$name}
循环


语法:
\{loop 变量名 输出数组 \}
　 //循环块
\{/loop\}

例如：
\{loop $students $std \}
　姓名:\{$std['name']\}
　年龄:\{$std['age']\}
　成绩:
　\{loop $std['score'] $score \} // 嵌套的循环
　　　\{$score['subject']\} : \{$score['score']\}
　\{/loop\}
\{/loop\}

 输出：

 {loop $students $std }
姓名:{$std['name']}
年龄:{$std['age']}
成绩:
{loop $std['score'] $score } 　　{$score['subject']} : {$score['score']}
{/loop}
{/loop}

HET支持在循环中使用continue与break，请看下一块的逻辑处理中的例子。
逻辑处理

语法:
\{if|elseif|while 条件\}
　statement...
{/if|while}
注：条件语体不需要用()括起来, if 或 elseif开头的以\{/if\}结束,\{while\}开始的则以\{/while\}结束

例如：
\{ $gender = 'female'\}{$gender = 'female'}
\{if $gender == 'male' \}
　He is Male.
\{elseif $gender == 'female'\}
　She is Female.
\{else\}
　Unknow.
\{/if\}

输出：{if $gender == 'male' }
　He is Male.
{elseif $gender == 'female'}
　She is Female.
{else} Unknow. {/if}

使用while的例子：
\{$i=0\}
\{while $i < 10 \}
　\{$i = $i+1\}
　\{$i\}
\{/while\}
 输出： {$i=0} {while $i < 10 } {$i = $i+1} {$i} {/while}

使用continue的例子(不输出5)：
\{$i=0\}
\{while $i < 10 \}
　\{$i = $i+1\}
\{if $i == 5 \}\{continue\}\{/if\}
　\{$i\}
\{/while\}
 输出： {$i=0} {while $i < 10 } {$i = $i+1} {if $i == 5 } {continue} {/if} {$i} {/while}

使用break的例子(不输出4以后的)：
\{$i=0\}
\{while $i < 10 \}
　\{$i = $i+1\}
\{if $i == 5 \}\{break}\{/if\}
　\{$i\}
\{/while\}
 输出： {$i=0} {while $i < 10 } {$i = $i+1} {if $i == 5 } {break} {/if} {$i} {/while}
使用函数

语法: \{ 函数名() \}

例如：\{myfunc()\}
输出：{myfunc()}

在其他语体中使用函数：
\{$date = date('Y-m-d')\} // date()函数赋值给$date
\{$date\}
{$date = date('Y-m-d')} 输出：{$date}

带参数使用： \{myfunc( $date )\}
输出：{myfunc($date)}

\{$number = 1234.567 \}{ $number = 1234.567 }
\{$numberf = number_format($number , 2 )\}{$numberf = number_format($number , 2 )}
\{$numberf\}
 输出：{$numberf}
赋值

语法: {变量名 = 值}

数字赋值：\{ $n = 123 \} {$n = 123}　　　　 输出：{$n}

字符串赋值：\{ $abc = 'abc' \} {$abc = 'I am abc'}　　　　 输出：{$abc}

变量赋值：\{ $ABC = $abc.' too , Value comes from $abc.' \}{$ABC = $abc.' too , Value comes from $abc.' }{$ABC = $abc.' too , Value comes from $abc.' }　　 输出：{$ABC}

注册函数赋值：\{ $date = date('Y-m-d H:i:s') \} {$date = date('Y-m-d H:i:s')}　　　　 输出：{$date}
include的使用

语法: \{include("文件名")\}
include的使用依赖于PHP程序的注册函数，注册了include后即可使用。请使用小写　

　　 例如：\{include("included.php")\}

结果：{include("included.php")}
$_REQUEST的使用

语法: 直接使用，键名不能有单引号
$_REQUEST的使用依赖于PHP程序。　

　　 例如：\{ $id = $_REQUEST["id"] \}{ $id = $_REQUEST["id"] } 在地址栏上加上?id=123输出\{$id\}

结果：{$id}
转义语体

语法: \\{ 内容 \\}

转义后将输出 { 内容 }，当且仅当需要输出正确的语体时才需要使用转义，转义意味着语体不会被编译。
